# デジタルログブック

## 概要
あなたの搭乗記録を手軽に、CAさんの手を煩わせることなく、永久に保存できます。
写真も掲載できます。
いつでもどこでも閲覧可能

## コンセプト
シンプルなログブック

## バージョン
Ruby 2.6.3
Rails 5.2.3

## 機能一覧
- ログイン機能	
- ユーザー登録機能	
- ログ一覧機能	
- ログ投稿機能	
- 写真投稿機能	
- ログ編集機能	
- ログ削除機能	
- マイページ	
- ログお気に入り機能	
- 検索機能
